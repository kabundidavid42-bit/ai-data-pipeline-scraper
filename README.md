# AI Data Pipeline Scraper

This project demonstrates a simple Python-based data pipeline for collecting, cleaning, and structuring data for downstream AI and analytics use.

## Features
- Web scraping using requests + BeautifulSoup
- Pagination support
- Data validation and cleaning
- Structured CSV output

## Use Case
Simulates real-world data collection workflows used in AI training and data preprocessing.

## Tech Stack
- Python
- requests
- BeautifulSoup
- pandas

## Run
pip install -r requirements.txt
python scraper.py

## Output
- Clean dataset exported as CSV
